## Funções autorizadas

### `print()`

A função `print()` é uma das funções mais básicas e utilizadas em Python. Ela serve para **exibir informações na tela** (saída padrão).

```python
print("Hello, Garden Community!")  # exibe um texto na tela
print(42)                           # também funciona com números
print("Valor:", 10)                 # pode receber múltiplos argumentos
```

Características importantes:

- Aceita qualquer tipo de dado: texto (`str`), números (`int`, `float`), variáveis, etc.
- Por padrão, adiciona uma **quebra de linha** ao final de cada chamada.
- Pode receber múltiplos argumentos separados por vírgula — eles serão exibidos com um espaço entre eles.

## Observações

- Escreva **apenas** a função, não um programa principal.
- Não chame a função diretamente no arquivo.
- Não escreva código fora da função.
